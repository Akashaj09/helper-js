# helper-js
sum simple helper method for js
# installation 
```
npm i aj-helper-js
```

# usage

```
  import { helperName } from 'aj-helper-js'
  
  const result = helperName(...parameter);
```

#Example

```
import { addDate } from 'aj-helper-js'
addDate(new Date(), 2);

```

# testing
Run
```
npm install
npm run build
npm run test
```
